🌐 Subdomain Enumerator

This is a Python script for enumerating subdomains of a given domain using DNS resolution. 🕵️‍♂️

✨ Features

✅ Resolves common subdomains using DNS queries.

✅ Filters valid subdomains based on DNS response.

✅ Handles exceptions like timeouts and non-existing domains.

✅ Provides a clean and structured output.

.

⚠️ Error Handling

❌ If a subdomain does not exist, it is silently ignored.

⏳ If a timeout occurs, the script will notify you.

⌨️ You can safely interrupt the script using CTRL+C.

📝 Notes

📂 The subdomain list is hardcoded; you can expand it by modifying the subdomain_array list.

🔒 This tool is for educational and security research purposes only. Ensure you have permission before scanning any domain.
